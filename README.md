# Customer Churn Analysis

## Project Overview

This project analyzes membership churn patterns by visualizing the relationship between:

* Days Since Membership Expiry
* Subscription Package
* Branch Location

The objective is to identify customer retention opportunities and understand churn behavior across different subscription plans.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Plotly
* Matplotlib

## Key Features

* Interactive scatter plot visualization
* Package-wise churn analysis
* Branch-level comparison
* Customer segmentation
* Data cleaning and preprocessing
* Statistical exploration of expiry patterns
